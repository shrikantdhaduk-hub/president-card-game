## Fastest hosting option: GitHub Pages
1. Create a new folder in your repo, for example `docs/`
2. Upload all files from this package into `docs/`
3. In GitHub: Settings → Pages
4. Source: Deploy from a branch
5. Branch: main
6. Folder: /docs
7. Save

Your site will publish at:
`https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/`

## Important
This version is a pass-the-phone game hosted on the web.
It is not real-time multiplayer across separate phones yet.
